## Installation

```bash
$ npm install
```

## Set-up the environment
1. Copy .env.example to .env
    ```bash
    $ cp .env.example .env
    ```
2. Edit your database configurations

## Running the migration

```bash
$ npm run migration:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
